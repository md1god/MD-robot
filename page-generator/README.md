## 🚀 البدء السريع

### 1️⃣ انسخ الملفات إلى مشروعك على GitHub

```bash
# انسخ المجلدات والملفات
cp pages-data.json your-repo/
cp build-pages.js your-repo/
cp -r .github/ your-repo/
```

### 2️⃣ اختياري: عدّل البيانات

افتح `pages-data.json` وأضف كلماتك الخاصة:

```json
{
  "contentWords": {
    "titles": [
      "عنوانك هنا",
      "عنوان آخر"
    ],
    "descriptions": [
      "وصفك هنا",
      "وصف آخر"
    ]
  }
}
```

### 3️⃣ فعّل GitHub Actions

```bash
# ادفع الملفات للـ GitHub
git add .
git commit -m "إضافة نظام مولد الصفحات"
git push origin main

# اذهب لـ: https://github.com/YOUR_USERNAME/YOUR_REPO/actions
# انقر "Enable workflows"
```

### 4️⃣ تشغيل يدوي (اختياري)

```bash
# في GitHub Actions UI انقر:
"Run workflow" → "Generate Pages Automatically"
```

## ⚙️ كيفية العمل

### دورة الحياة:

1. **كل ساعة** - GitHub Actions يستدعي الـ workflow
2. **Script يعمل** - `build-pages.js` ينشئ صفحة جديدة
3. **محتوى عشوائي** - يختار من كلماتك عشوائياً
4. **تحديث الحالة** - `build-state.json` يسجل التقدم
5. **Git Push** - الملفات الجديدة ترفع للـ GitHub

### مثال على الدورة:

```
الساعة 1:00 → إنشاء page-0001.html ✅
الساعة 2:00 → إنشاء page-0002.html ✅
الساعة 3:00 → إنشاء page-0003.html ✅
...
الساعة 500:00 → وصلنا للحد! إعادة تعيين 🔄
الساعة 501:00 → إنشاء page-0001.html (دورة جديدة) ✅
```

## 🎨 تخصيص المحتوى

### تغيير الألوان:

```json
"colors": [
  "#c9a84c",  // ذهبي
  "#f0c060",  // ذهبي فاتح
  "#8b1a1a",  // أحمر دم
  "#e8d5a3"   // فاتح
]
```

### إضافة عناوين جديدة:

```json
"titles": [
  "عنواني الجديد",
  "عنوان آخر مثير",
  "شيء فريد"
]
```

### تغيير وقت التشغيل:

عدّل `.github/workflows/generate-pages.yml`:

```yaml
on:
  schedule:
    - cron: '0 */6 * * *'  # كل 6 ساعات
    # أو
    - cron: '0 0 * * *'    # مرة في اليوم
    # أو
    - cron: '*/30 * * * *' # كل 30 دقيقة
```

## 🐛 استكشاف الأخطاء

### المشكلة: الـ Workflow لا يعمل

✅ **الحل:**
1. تأكد أن `.github/workflows/generate-pages.yml` في الـ repo
2. انقر **Actions** في GitHub وفعّل الـ workflows
3. اختبر: **Run workflow** يدوياً

### المشكلة: لا توجد ملفات جديدة

✅ **الحل:**
1. تحقق من GitHub Actions logs: **Actions** → آخر run
2. تأكد أن `build-pages.js` صحيح
3. جرّب محلياً: `node build-pages.js`

### المشكلة: الـ Pages لا تظهر صحيح

✅ **الحل:**
1. تحقق من أن `pages-data.json` JSON صحيح (استخدم JSONLint)
2. عدّل الألوان في `build-pages.js` إذا لم تظهر
